# SporeScraper
A tool for data analysis and visualization of the contents of the spore.fandom.com page,
made as a final project for the university Python course.

The project was made in accordance with the CC-BY-SA licence on the fandom website.

### How to run
Create a Python virtual environment and install dependencies in the folder where the program files have been downloaded.
```
python3 -m venv wikiScraperEnv
source wikiScraperEnv/bin/activate
python3 -m pip install -r requirements.txt
```
Then input `python3 wiki_scraper.py [put arguments here]` to run.  
Type in `python3 wiki_scraper.py -h` for more information about how to run the program.
